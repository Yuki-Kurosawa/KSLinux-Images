# KSLinux-Images
Docker images for Yuki's Linux distro (KSLinux) series

Started from KSLinux 26 , old versions from [YukiDocker](https://github.com/YukiKurosawaDev/YukiDocker) is deprecated and archived.

# Versions
------------
KSLinux 26: it will release at April,2026. (LTS, 2-year updates by Yuki & 8-year updates by community)<br/>
KSLinux 25.10: it is released at October 10,2025 (9-month updates by community)<br/>
KSLinux 24.04: it is released at April 26,2024 (LTS, 2-year updates by Yuki & 8-year updates by community)<br/>

# Update Logs
------------
1. KSLinux 25.10 breaks sudo and coreutils uses due to the replacement for sudo and coreutils to sudo-rs and uutils.<br/>
There is no warranties for your scripts will work properly.<br/>
2. KSLinux 26 may break many programs due to KSLinux 26 rebases to another distro base, so the version format of KSLinux also changes.<br/>
